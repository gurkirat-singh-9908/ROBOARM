6 DOF Robotic Arm Control System

About the Robotic Arm

This is a 6 DOF (degrees of freedom) articulated robotic arm with a spherical wrist, designed for precision control and automation tasks.

This web interface allows users to control a real hardware model through various features, providing both manual and automatic control options.

Specifications:

  • Total arm length: 0.8 meters
  • Payload capacity: 600 grams
  • Type: Articulated with spherical wrist
Control Methods

Manual Control: Use sliders and knobs to precisely position the robotic arm in 3D space.

Automatic Control: Select objects from a trained list for the arm to automatically locate and pick up, then use manual control for placement.
